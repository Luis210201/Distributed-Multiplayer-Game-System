## Purpose

This project involves the interaction between various clients, including wasps, roaches, and lizards, with a central server. The lizards have the ability to move freely while adhering to certain restrictions and can also consume other types of clients.

## Server Setup

Execute make in the terminal to compile the server.
Run the server by executing ./server.

## Lizard Client

Execute make in the terminal to compile the Lizard client.
Run the Lizard client by executing ./lizard <your_ip_address> 5555 5558.

## Roach Client

Execute make in the terminal to compile the Roach client.
Run the Roach client by executing ./roach <your_ip_address> 5557.

## Wasp Client

Execute make in the terminal to compile the Wasp client.
Run the Wasp client by executing ./wasp <your_ip_address> 5557.

Note: Replace <your_ip_address> with the actual IP address of your computer.
